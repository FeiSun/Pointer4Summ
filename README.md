# Pointer4Summ
Pointer network for extractive summarization



## Usage

**Requirements**

To run the souce codes, some external packages are required

* python 2.7
* Tensorflow  1.4

**Input**

data file:
```bash
<original title>\t\t<short title>
```

vocab file:
```bash
<word>\t<count>
```
